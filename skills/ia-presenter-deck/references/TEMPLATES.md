# iA Presenter Templates

These are rhetorical templates, not rigid slide counts.

## 1. Pitch Deck

1. cover
2. problem reframed
3. what people actually feel
4. contrast or comparison
5. what changes
6. proof or pilot
7. why now
8. decision / next step

Good surface mix:
- sharp cover
- quote or customer-voice slide
- one crisp comparison
- one tiny proof table
- one closing landing line

## 2. Board / Executive Update

1. short version up front
2. what changed
3. what we stopped doing
4. what we doubled down on
5. where we were wrong
6. risks we are not hiding
7. what we need
8. what success looks like next

Good surface mix:
- short-version slide
- one before / after or stop / start contrast
- one exact proof table
- one explicit ask slide

## 3. Workshop / Teaching Deck

1. what this session gives you
2. the first mistake
3. the model or ladder
4. weak example / strong example
5. how to apply it
6. common traps
7. practice / action

Good surface mix:
- agenda
- weak / strong contrast
- ladder or sequence slide
- practice or checklist slide

## 4. Case Study / Story Deck

1. scene
2. real problem
3. before / after
4. what changed first
5. what we refused to do
6. result
7. what surprised us
8. what you can copy
9. closing line

Good surface mix:
- scene-setting cover or image
- before / after contrast
- one refusal or principle slide
- one visible takeaway line

## Reusable Slide Patterns

### Reframe

Use when the audience thinks the problem is one thing, but you need to shift the lens.

### Symmetry

Use one line on each side only when the contrast is genuinely crisp.

### One-Line Landing

Use for a closing or turning-point slide. One short line on slide. The argument lives in notes.

### Proof Table

Use only when a table is the fastest way to show movement. Keep it tiny.

### Sequence

Use when order matters. Prefer 3-4 steps, not 8.
