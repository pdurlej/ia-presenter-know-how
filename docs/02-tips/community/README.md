# Community Tips

This directory is for community-contributed tips and best practices for iA Presenter.

## How to Contribute

See the main [README.md](../../README.md#-share-your-best-practices) for detailed instructions on sharing your tips.

## Tips Format

Each tip should be a markdown file following this structure:

```markdown
# Your Tip Title

> **Use case:** Briefly describe when to use this tip

## What It Solves

Explain what problem or challenge this tip addresses.

## How It Works

Provide step-by-step explanation or example:

\`\`\`markdown
## Example Slide

	This content shows the technique
\`\`\`

## Why This Works

Explain why this approach is effective.

## Tips & Variations

- Variation 1
- Variation 2
- Common pitfalls to avoid
```
